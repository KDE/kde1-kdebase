2. KFM Configuration: Color

In this screen you can customize the File Manager color settings. When you click on a color box, the "Select Color"screen allows you to select a color for the corresponding item.

  • "Background Color" is the color of the background in a File Manager window.
  • "Normal Text Color is the color used for displaying normal text (i.e., text without hypertext links) within a hypertext document viewed with the KDE File Manager and Browser, KFM.
  • "URL Link Color" is the color in which clickable hypertext links are shown on pages viewed with the KDE File Manager and Browser (KFM), before the link is followed. Directory entries viewed with KFM are also URL hypertext links.
  • "Followed Link Color" is the color in which clickable links are shown, after the link has been followed, during the period that this event is remembered.

Two further buttons control how links are displayed:

  • If "Change cursor over link" is selected, the cursor changes from an arrow to a pointing hand when it is over a clickable link.
  • If "Underline links" is selected, clickable links are underlined.

The final button allows the user's default color scheme to be forced;

  • If "Always use my colors" is selected, the user's default color scheme will override any colors specified by the web page.
